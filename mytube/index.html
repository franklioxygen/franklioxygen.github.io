Quick start

From Docker pull to first download in minutes.

1

Docker (recommended)

Pull the official GHCR image and expose port 5551.

docker run -d \
  --name mytube \
  -p 5551:5551 \
  -v ./data:/app/data \
  -v ./uploads:/app/uploads \
  ghcr.io/franklioxygen/mytube:latest
2

From source

Clone the repo, install dependencies, and run the dev servers.

git clone https://github.com/franklioxygen/mytube.git
cd mytube
npm run install:all
npm run dev
3

Open the UI

Visit localhost:5551, paste a video URL, and hit download.

Requirements

  • Node.js 20–26 — for local development (Docker ships Node 22)
  • yt-dlp + ffmpeg — required for downloads and transcoding
  • Python 3.8+ — for yt-dlp and PO Token provider plugin
